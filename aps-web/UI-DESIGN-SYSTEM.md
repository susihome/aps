## 设计原则总结

### 企业级 Dashboard 特点
1. **数据优先**：最大化数据可见性，最小化装饰
2. **高效布局**：紧凑但不拥挤，清晰的视觉层次
3. **专业配色**：蓝色系传达信任和专业
4. **精确字体**：等宽字体用于数据，无衬线字体用于 UI
5. **一致性**：统一的圆角、阴影、间距系统

### 避免的反模式
- ❌ 过度装饰（渐变、阴影、动画）
- ❌ 混合多种设计风格
- ❌ 使用 emoji 作为图标
- ❌ 过大的间距（浪费空间）
- ❌ 不一致的配色
