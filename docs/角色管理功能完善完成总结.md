## 🚀 使用说明

### 访问角色管理页面
1. 登录系统（需要 ADMIN 角色）
2. 点击左侧菜单 "系统管理" → "角色管理"
3. 进入角色管理页面

### 创建角色
1. 点击 "新增角色" 按钮
2. 填写角色名称和描述
3. 点击 "保存"

### 分配权限
1. 在角色列表中点击 "分配权限"
2. 在权限树中选择需要的权限
3. 点击 "保存"

### 批量操作
1. 选中多个角色
2. 点击 "批量删除" 或 "批量分配权限"
3. 确认操作
