# 角色管理快速测试指南

## 🚀 快速开始

### 前置条件
- ✅ 后端已启动
- ✅ 前端已启动
- ✅ 已登录系统（admin 用户）

---

## 📝 快速测试清单

### 1️⃣ 页面加载测试 (1 分钟)

```
操作: 点击左侧菜单 "系统管理" → "角色管理"

验证:
✅ 页面正常加载
✅ 显示三个角色卡片（ADMIN、PLANNER、SUPERVISOR）
✅ 每个角色显示不同的图标和颜色
✅ 右侧显示权限配置面板
✅ 权限树正确加载
```

---

### 2️⃣ 角色图标和颜色测试 (1 分钟)

```
验证:
✅ ADMIN 角色: 红色渐变图标 + 用户图标
✅ PLANNER 角色: 蓝色渐变图标 + 文档图标
✅ SUPERVISOR 角色: 绿色渐变图标 + 锁图标
✅ 每个角色显示对应的中文名称
```

---

### 3️⃣ 权限配置测试 (3 分钟)

```
操作:
1. 选择 PLANNER 角色
2. 在权限树中勾选 3-5 个权限
3. 点击"保存"按钮
4. 验证保存成功提示

验证:
✅ 权限树正确显示
✅ 可以勾选/取消权限
✅ 保存成功显示提示
✅ 已选择计数正确更新
```

---

### 4️⃣ 全选/清空测试 (1 分钟)

```
操作:
1. 点击"全选"按钮
2. 验证所有权限被选中
3. 点击"清空"按钮
4. 验证所有权限被取消

验证:
✅ 全选功能正常
✅ 清空功能正常
✅ 计数正确更新
```

---

### 5️⃣ 搜索功能测试 (1 分钟)

```
操作:
1. 在搜索框输入 "ADMIN"
2. 验证列表只显示 ADMIN
3. 清除搜索框
4. 验证显示所有角色

验证:
✅ 搜索功能正常
✅ 搜索结果正确
```

---

### 6️⃣ 角色编辑测试 (2 分钟)

```
操作:
1. 点击 PLANNER 角色右侧"..."菜单
2. 选择"编辑"
3. 修改描述
4. 点击"保存"

验证:
✅ 编辑对话框打开
✅ 角色名称禁用
✅ 描述可编辑
✅ 编辑成功
```

---

### 7️⃣ 权限持久化测试 (2 分钟)

```
操作:
1. 为 SUPERVISOR 角色分配权限
2. 点击"保存"
3. 刷新页面 (F5)
4. 再次选择 SUPERVISOR 角色
5. 验证权限仍然被选中

验证:
✅ 权限保存成功
✅ 刷新后权限仍然保留
```

---

### 8️⃣ 错误处理测试 (1 分钟)

```
操作:
1. 尝试删除 ADMIN 角色
2. 查看错误提示

验证:
✅ 显示错误提示
✅ 错误信息清晰
```
